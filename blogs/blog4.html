Node.js Modules

Beginner Guide to Node.js fundamentals

Node.js modules are reusable blocks of code that help developers organize their applications into smaller, manageable parts. Instead of writing all code in a single file, modules allow you to separate functionality into different files and reuse them whenever needed.

Node.js comes with many built-in modules such as HTTP, File System, Path, and Events. Developers can also create their own modules or use third-party modules from npm.

---

Types of Modules in Node.js

Node.js supports three types of modules:

  • Core Modules: Built-in modules provided by Node.js (e.g., HTTP, FS)
  • Local Modules: Custom modules created by developers
  • Third-party Modules: Installed using npm (e.g., Express)
---

Core Modules Example

Core modules are included with Node.js, so you don't need to install them separately. 

const http = require('http');

http.createServer((req, res) => {
    res.write("Hello from core module");
    res.end();
}).listen(3000);

In this example, the HTTP module is used to create a simple web server.

---

Local Modules Example

Developers can create their own modules to reuse code.

Step 1: Create a file (math.js)

exports.add = function(a, b) {
    return a + b;
};

Step 2: Use it in another file

const math = require('./math');

console.log(math.add(5, 3));

This allows you to separate logic into different files and reuse it easily.

---

Third-party Modules

Third-party modules are installed using npm and provide additional functionality. 

npm install express

Example using Express: 

const express = require('express');
const app = express();

app.get('/', (req, res) => {
    res.send("Hello from Express");
});

app.listen(3000);
--- ---

Why Modules are Important

  • Improves code organization
  • Promotes reusability
  • Makes debugging easier
  • Reduces code duplication
---

How require() Works

The require() function is used to include modules in Node.js. It loads the module and makes its functionality available for use in another file. 

const fs = require('fs');

Here, the File System module is imported and can be used to work with files.

---

Module Exporting Methods

Node.js provides different ways to export modules:

  • exports
  • module.exports

Example:

module.exports = function() {
    console.log("Hello Module");
};
---

Conclusion

Modules are one of the most important features of Node.js. They allow developers to structure applications efficiently and reuse code across multiple files. By using modules, developers can build scalable and maintainable applications.
