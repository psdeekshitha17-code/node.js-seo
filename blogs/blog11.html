REST API

Advanced Guide to Node.js fundamentals

Overview

REST API allows communication between client and server using HTTP methods.

HTTP Methods

  • GET – Fetch data
  • POST – Create data
  • PUT – Update data
  • DELETE – Remove data

Example

app.get('/users', (req,res)=>res.send("Users data"));

Conclusion

REST APIs are essential for modern web applications.
