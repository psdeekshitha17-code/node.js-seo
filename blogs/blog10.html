MIDDLEWARE IN EXPRESS.js

Intermediate Guide to Node.js fundamentals

Overview

Middleware functions are used to process requests before sending a response.

Types of Middleware

  • Application-level
  • Router-level
  • Error-handling

Example

app.use((req,res,next)=>{
console.log("Middleware");
next();
});

Conclusion

Middleware plays a key role in request processing.
