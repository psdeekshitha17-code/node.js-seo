CREATING SERVER USING HTTP

Intermediate Guide to Node.js fundamentals

Overview

Node.js provides the HTTP module to create web servers easily.

Steps to Create Server

  • Import HTTP module
  • Create server
  • Listen on port

Example

const http = require('http');
http.createServer((req, res) => {
res.write("Hello World");
res.end();
}).listen(3000);

Conclusion

Node.js makes it easy to create lightweight servers.
