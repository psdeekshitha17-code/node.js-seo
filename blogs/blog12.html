MongoDB

Advanced Guide to Node.js fundamentals

Overview

MongoDB is a NoSQL database used with Node.js for storing data.

Steps

  • Install MongoDB
  • Install mongoose
  • Connect database

Example

mongoose.connect("mongodb://localhost:27017/db");

Conclusion

MongoDB is widely used with Node.js for data storage.
