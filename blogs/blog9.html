ROUTING IN EXPRESS.js

Intermediate Guide to Node.js fundamentals

Overview

Routing refers to how an application responds to client requests.

Types of Routing

  • GET
  • POST
  • PUT
  • DELETE

Example

app.get('/home', (req,res)=>res.send("Home Page"));

Conclusion

Routing helps manage different endpoints in an application.
