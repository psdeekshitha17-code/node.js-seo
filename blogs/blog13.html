CRUD Operations

Advanced Guide to Node.js fundamentals

Overview

CRUD stands for Create, Read, Update, Delete.

Operations

  • Create – Add data
  • Read – Retrieve data
  • Update – Modify data
  • Delete – Remove data

Example

app.post('/add', ...);
app.get('/view', ...);

Conclusion

CRUD operations are fundamental in any application.
